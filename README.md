# Adeshola Faith | Developer Portfolio

A modern, responsive portfolio website showcasing my work as a **Frontend Developer** and  **Developer Advocate**. The project was designed to reflect my approach to building accessible, high-performance web experiences while highlighting my projects, technical skills and community building.

**Live Demo:** https://adesholafaith.github.io

* Responsive design for desktop, tablet, and mobile
* Modern UI with smooth animations and scroll reveals
* Interactive project showcase
* Skills and experience timeline

# Code Architecture

## HTML

The project uses semantic HTML elements to improve accessibility, SEO and maintainability.

Examples include:

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<footer>`

Each major section of the portfolio is separated into logical components such as:

* Hero
* About
* Experience
* Skills
* Projects
* Contact

---

## CSS

The styling follows a component-based approach.

Key techniques include:

* CSS Variables for theme consistency
* Flexbox
* CSS Grid
* Responsive Media Queries
* Hover Animations
* Custom Buttons
* Gradient Borders
* Reusable Utility Classes

The design emphasizes readability, spacing and visual hierarchy while remaining lightweight.

---

## JavaScript

JavaScript enhances user interaction throughout the portfolio.

Main responsibilities include:

* Navigation interactions
* Scroll reveal animations
* Active navigation highlighting
* Contact form submission using EmailJS
* Form validation
* Loading states
* Success and error feedback
* Smooth scrolling

The code is organized into reusable functions to keep the project modular and easy to maintain.

---

# Contact Form

The contact form is powered by **EmailJS**, allowing visitors to send messages directly from the website without requiring a backend server.

Features include:

* Required field validation
* Loading state while sending
* Success and error feedback
* Automatic form reset after successful submission

# Getting Started

Clone the repository:

```bash
git clone https://github.com/adesholafaith/adesholafaith.github.io.git
```

Navigate into the project:

```bash
cd adesholafaith.github.io
```

Open `index.html` in your browser or use a local development server such as VS Code Live Server.
